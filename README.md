# SmartRockets

Genetic algorithm-powered rockets learning to navigate obstacles.

## Overview

A refactored implementation of Daniel Shiffman's Smart Rockets tutorial, applying functional programming principles to improve code clarity. Rockets evolve over generations to find optimal paths to a target while avoiding obstacles.

## Features

- Genetic algorithm with fitness-based selection
- Real-time visualization with p5.js
- Obstacle collision detection
- Performance profiling and statistics
- Modular ES6 code structure

## Prerequisites

- Modern web browser
- [Brackets](http://brackets.io) or any web server (for local development)

## Run

1. Open the `SmartRockets` folder in Brackets
2. Open `source/index.html`
3. Click the live preview button

Or serve the files with any local web server:

```bash
cd source
python -m http.server 8000
# Open http://localhost:8000
```
